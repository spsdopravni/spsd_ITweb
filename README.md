# 🎓 Školní rozcestník pro žáky

Umožňuje žákům snadný a rychlý přístup ke všem školním digitálním systémům a informacím na jednom místě – přehledně, intuitivně a bez nutnosti hledat odkazy jinde.

---

## 🚀 Tech Stack

- **React** – frontendová knihovna
- **Tailwind CSS** – utility-first framework pro styling
- **Lucide Icons** – moderní ikony v Reactu
- **Framer Motion** – animace a přechody

---

```bash
git clone https://github.com/spsdopravni/spsd_studentDashboard.git
